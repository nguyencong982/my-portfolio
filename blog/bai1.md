# Bài 01: Java Basic - Cú pháp và Kiểu dữ liệu

Java là ngôn ngữ lập trình hướng đối tượng. Để bắt đầu, chúng ta cần nắm vững các kiểu dữ liệu cơ bản:

### 1. Các kiểu dữ liệu nguyên thủy
* **int**: Kiểu số nguyên (Ví dụ: 1, 2, 3)
* **double**: Kiểu số thực (Ví dụ: 5.5, 10.2)
* **boolean**: Kiểu đúng/sai (true/false)

### 2. Biến trong Java
Khai báo biến theo cú pháp: `[Kiểu dữ liệu] [Tên biến] = [Giá trị];`
Ví dụ: `int age = 20;`
