# Bài 07: Đa luồng (Multithreading) trong Java

Đa luồng cho phép thực hiện nhiều tác vụ cùng một lúc để tối ưu hiệu suất.

### 1. Tạo luồng
* Cách 1: Kế thừa lớp `Thread`.
* Cách 2: Triển khai Interface `Runnable`.

### 2. Các trạng thái của luồng
New, Runnable, Running, Blocked, Waiting, Terminated.
