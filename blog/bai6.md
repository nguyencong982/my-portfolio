# Bài 06: Java Collections Framework

Collections giúp quản lý nhóm các đối tượng hiệu quả hơn mảng.

### 1. List (ArrayList, LinkedList)
Dòng danh sách có thứ tự, cho phép trùng lặp.

### 2. Set (HashSet)
Danh sách không chứa các phần tử trùng lặp.

### 3. Map (HashMap)
Lưu trữ dữ liệu theo cặp Key - Value.
